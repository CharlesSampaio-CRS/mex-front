# Assets

Coloque aqui os recursos estáticos da aplicação:

## Estrutura sugerida:

### /icons
- Ícones do aplicativo (app.icns para macOS, app.ico para Windows, etc.)
- Ícones da interface

### /images
- Logos das exchanges
- Imagens de fundo
- Gráficos estáticos

### /fonts (opcional)
- Fontes customizadas se necessário

## Gerando ícones do app

Para gerar ícones para todas as plataformas, você pode usar:
- [electron-icon-builder](https://www.npmjs.com/package/electron-icon-builder)
- [electron-icon-maker](https://www.npmjs.com/package/electron-icon-maker)

Ou use ferramentas online como:
- https://icon.kitchen/
- https://www.img2go.com/convert-to-ico
